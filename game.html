p5,js Game!!


I wanted my game to involve something with eating/collecting an object to gain points and I realized I can have it be Pikmin themed! Instead of having it be level based I just added a timer that shows a different message if you ate all the Pikmin or not. I originally had the timer start at 30 seconds but that was a little to easy. My game ended up feeling like those flash games you would stumble upon when trying to find a game on the computer that advertises itself as the real game. Over all this was really fun to do!
